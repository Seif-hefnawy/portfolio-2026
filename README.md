# 🚀 Personal Portfolio 2026 |  

بورتفوليو شخصي عصري ومستجيب (Responsive) مبني باستخدام تقنيات الـ Frontend الأساسية، مع نظام تغيير ثيمات متقدم ودعم للوضع الليلي.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5 & CSS3**: للهيكلة والتنسيق المتقدم.
* **Tailwind CSS**: للتصميم السريع والمستجيب باستخدام الـ Utility Classes.
* **Vanilla JavaScript**: للتحكم في المنطق (Logic) وتغيير الثيمات وتفاعل المستخدم.
* **Font Awesome**: للأيقونات التفاعلية.
* **Google Fonts**: خطوط Alexandria و Tajawal لتجربة قراءة عربية مميزة.

## ✨ المميزات (Features)

- 🎨 **Multi-Theme Switcher**  
  اختيار بين عدة ألوان (أزرق – أخضر – بنفسجي – برتقالي)

- 🌙 **Dark / Light Mode**  
  دعم كامل للوضع الليلي والنهاري

- 📱 **Responsive Design**  
  متوافق مع جميع الشاشات (موبايل – تابلت – ديسكتوب)

- ⚡ **Smooth Animations**  
  انتقالات سلسة باستخدام CSS

- 🧠 **Dynamic Portfolio Filter**  
  فلترة المشاريع مع أنيميشن (Scale & Opacity)

- 🎯 **Interactive UI Components**
  - Tabs تفاعلية
  - Carousel Indicators ذكية (Active State)
  - منع القفز المفاجئ (Smooth Navigation)

- 💾 **Local Storage**  
  حفظ إعدادات المستخدم (الثيم + الوضع الليلي)
