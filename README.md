# 🚀 Personal Portfolio 2026 |  ahmed El-Hefnawy

بورتفوليو شخصي عصري ومستجيب (Responsive) مبني باستخدام تقنيات الـ Frontend الأساسية، مع نظام تغيير ثيمات متقدم ودعم للوضع الليلي.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5 & CSS3**: للهيكلة والتنسيق المتقدم.
* **Tailwind CSS**: للتصميم السريع والمستجيب باستخدام الـ Utility Classes.
* **Vanilla JavaScript**: للتحكم في المنطق (Logic) وتغيير الثيمات وتفاعل المستخدم.
* **Font Awesome**: للأيقونات التفاعلية.
* **Google Fonts**: خطوط Alexandria و Tajawal لتجربة قراءة عربية مميزة.

## ✨ المميزات (Features)

* **Multi-Theme Switcher**: إمكانية الاختيار بين 4 ألوان مختلفة (الأزرق، الأخضر، البنفسجي، والبرتقالي الناري).
* **Dark/Light Mode**: دعم كامل للوضع الليلي والنهاري مريح للعين.
* **Responsive Design**: متوافق تماماً مع جميع الشاشات (موبايل، تابلت، ديسكتوب).
* **Smooth Animations**: انتقالات وحركات ناعمة باستخدام CSS Transitions.
* **Local Storage**: حفظ اختيار المستخدم للثيم والوضع الليلي بحيث لا يتغير عند تحديث الصفحة.
